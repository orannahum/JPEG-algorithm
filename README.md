# JPEG-algorithm


Jpeg compression.
1.Change the color channel RGB image to YCbCr Image

2.Transform the image into blocks 8 * 8

3.DCT

4.Zig-Zag process

and Huffman Coding

4.Saving as jpeg, do step-5 for all blocks of the image

5.Saving as jpeg

6.Decoding
